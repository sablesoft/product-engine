# World Bootstrap Checklist

When creating the initial adventure base, ensure all of the following exist:

- adventure folder under `adventures/<slug>/`
- canon/setting.md
- canon/global_story.md
- canon/tone_and_themes.md
- canon/rules_of_world.md
- at least 3 locations
- at least 2 active quests
- at least 1 event file
- state/runtime.yaml
- state/campaign_state.yaml
- state/facts.yaml
- state/flags.yaml

Do not create player character files in this skill.
Set next phase to `bootstrap_character`.
